# بوت الذكاء الاصطناعي المخصص: المؤرخ المصري

تهانينا! لقد تم تصدير البوت الخاص بك بنجاح من منصة **Hathor**.

## محتويات الملف المصدّر:
1. `config.json`: يحتوي على إعدادات البوت والتعليمات البرمجية التوجيهية (System Prompt) والألوان المخصصة له.
2. `index.html`: صفحة ويب مستقلة وجذابة لفتح شات مع البوت وتجربته فوراً في متصفحك.
3. `server.js`: خادم Node.js متكامل ومجهز برباط متكامل بالمنصة المستهدفة (TELEGRAM).
4. `package.json`: ملف الاعتمادات والمكتبات البرمجية اللازمة للتشغيل.

## طريقة التشغيل الفوري لمحاكاة الويب:
فقط افتح ملف `index.html` في أي متصفح لديك وستتمكن من اختبار البوت مباشرة!

## طريقة تشغيل كود خادم دمج القنوات (TELEGRAM):
1. تأكد من تثبيت بيئة عمل [Node.js](https://nodejs.org).
2. افتح مجلد البوت في موجه الأوامر (Terminal).
3. قم بتثبيت المكتبات البرمجية بتشغيل الأمر:
   ```bash
   npm install
   ```
4. قم بفتح ملف `server.js` والتأكد من إدخال مفاتيح التوكن (Tokens) والربط بشكل صحيح.
5. شغل الخادم بالأمر التالي:
   ```bash
   npm start
   ```

شكراً لاستخدامك منصة **Hathor**!
